## Cómo actualizar los datos
1. Abrí Claude (claude.ai)
2. Subí el Excel `Informe.xlsx` + el archivo `tablero_agro.html`
3. Escribí: 
"Actualizá el tablero con este Excel. Solo usar solapas Cereales, TC e Inflación. El HTML actual es este archivo. Extraer solo estos datos frescos y reemplazarlos en el HTML:

FUT_RAW y FUT_HIST: precios futuros por posición
PIZ_MAP: última pizarra por cultivo
FAS: último FAS teórico por cultivo
TC_SPOT y TC_FUT: tipo de cambio spot y futuros Rofex
INF: inflación REM próximos 12 meses
DISP_MAP: disponibles desde julio 2025 en adelante (clave mes_año) No tocar: DISP_HIST, HIST_COSECHA, EVOL_DISP, EVOL_COSECHA, PRECIO_COSECHA_REF (series históricas largas — solo actualizar si hay datos nuevos significativos)."


4. Descargá el HTML nuevo y subilo acá reemplazando el anterior
5. Esperá 1-2 minutos y recargá el link de arriba
